Ingeniería de Software con AI

Desarrollador full-stack que utiliza AI como co-piloto para construir y mantener sistemas reales en producción. Más allá de "usar ChatGPT", diseña arquitectura, integra servicios, opera infraestructura y resuelve problemas concretos del día a día.

Aplicaciones en Producción

  • Finance Dashboard (v7.8.x) — Dashboard financiero familiar con planeación, multi-moneda MXN/USD y tracking de Cetes/Bitcoin. Vanilla JS + Vite, backend serverless en Cloudflare Workers, Google Sheets como fuente de datos. Desplegado en GitHub Pages, en uso diario desde hace más de 8 meses.
  • Manager App (v1.1.x) — App mobile-first para gestión de catálogo musical y carrera. React + Howler.js, Worker en Cloudflare como backend, Notion como base de datos, OAuth de Google y streaming seguro de audio desde Google Drive vía Service Account.
  • Finance v2 API Server — API REST en TypeScript (Fastify + zod + sql.js) que alimenta Dashboard y Manager App. Daemonizada en macOS con LaunchAgent (KeepAlive), credenciales en macOS Keychain (sin archivos `.env` en el repositorio).
  • Finance MCP Server — Servidor MCP (Model Context Protocol) en Node.js + Fastify, desplegado en Fly.io. Doble transporte HTTP + stdio, consumido simultáneamente por Claude, Gemini y widgets nativos de iOS.
  • Elussila Gallery — Galería de arte en Next.js 16 + Firebase, con flujo de upload vía OAuth de Google Drive. Desplegado en Vercel.
  • MK Composer (v1.0.x) — PWA de asistencia a composición con Groq/LLaMA, OAuth de Google y Google Sheets como backend de catálogo.

Infraestructura AI y Sistemas de Agentes

  • Memoria persistente multi-AI (Engram) — Sistema SQLite con observaciones, *topic keys* y relaciones (supersedes / conflicts / scoped) compartido entre Claude Code, Gemini, OpenCode y ChatGPT a través de máquinas distintas. Permite a cualquier agente recuperar el contexto y decisiones de cualquier otro.
  • +43 *skills* modulares de agente — Biblioteca propia de capacidades para Claude Code y Antigravity con delegación a sub-agentes, orquestación y cambio de contexto controlado.
  • Spec-Driven Development end-to-end — Flujo explorar → proponer → especificar → diseñar → tareas → implementar → verificar → archivar, con artefactos persistidos en memoria y trazabilidad por cambio.
  • Coordinación multi-modelo — Claude + Gemini + OpenCode operando sobre un mismo SQLite como fuente única de verdad, sin reentrenamientos, sin servicios cloud propietarios.

Patrones de Ingeniería Aplicados

  • *Evidence-first ledger* — toda transacción financiera obliga a un artefacto verificable antes de registrarse.
  • *Stateless secrets* — credenciales en macOS Keychain, nunca en archivos del repositorio.
  • *Webhook bridges* event-driven para sincronización entre servicios.
  • Arquitectura híbrida: edge compute (Cloudflare) + serverless + SQLite como backend persistente.
  • OAuth completo (Google Drive, YouTube, Gmail, Calendar) tanto en navegador como en backend.
